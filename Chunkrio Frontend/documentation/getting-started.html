Getting Started

To access the splitting tool, you will need to sign up with Chunkr.iO. This would give you access to a storage space to retain your files for later use. Chunkr.iO splitting tool supports two file formats which are CSV files and JSON files.

To begin, please click on the button below to sign up and be provided access to your user dashboard.

Account and Profile

Chunkr.iO provides users with an account and allows customization of profile data anytime. The user is prompted to protect account data using a password. User passwords can be reset in the case that user loses password or decides to change it.

Accessing your dashboard

Users can view their dashboard after signing-up and hold full access to their own library of files.

Easy access of files

On the "All files" section of the user dashboard, users can quickly find and follow up with recently splitted files and can decide to re-split, download or delete such files. Under "Easy Access", you can preview up to 4 recent files splitted in the last 24 hours.
