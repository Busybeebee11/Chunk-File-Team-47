How to split files

For an unauthenticated user, you would have to:

  • Sign up/ Create an account
  • Navigate to your dashboard
  • Click on the “Split” button and select the type of file you want to split
  • Upload your file and choose the number of files you want to split it to and split.
  • The splitter is going to return your chunked files to your dashboard
  • Download and use or save to download later. 🙂

For an authenticated user, you would have to:

  • Sign in to your account
  • Navigate to your dashboard
  • Click on the “Split” button and select the type of file you want to split
  • Upload your file and choose the number of files you want to split it to and split.
  • The splitter is going to return your chunked files to your dashboard
  • Download and use or save to download later.🙂
