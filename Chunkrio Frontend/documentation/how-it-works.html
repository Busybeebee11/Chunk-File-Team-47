How the tool works

Chunkr.iO splitting tool allows splitting by letting users split with a parameter that allows users to select their preferred split size. The splitter supports CSV and JSON file formats.

Due to the tool still in it's early stage of functioning, the tool limits the size of the file to be uploaded for splitting. Therefore, users can only split a certain size of file if it falls below the limit.

The splitter is able to recognize its supported file formats and would show an error message to the user if the uploaded file is not a CSV or JSON file format. This helps protect the tool from getting corrupted by malicious content and helping the user stay aware of the type of file they are about to split.

With Chunkr.iO's storage provision to authenticated users, registered users can either choose to save or download the chunked files. If they save it, files will be stored in the database and in the case that a user doesn't save their splitted files, it'll stay visible for a total of 24 hours before deletion.
