Introduction

Chunkr.iO is a splitter tool service that provides users with the ability to selectively divide CSV and JSON files using chunk size as a parameter for division. With extensive research and idea implementation, Team 47 aims to deliver an optimised method of splitting, ensuring that shortcomings like inaccuracy, system errors and data incompletion are non-existent.

What is a CSV file

A CSV (Comma-Separated Value) file, as the name implies, typically separates information using commas. It's a way to exchange structured information, like the contents of a spreadsheet, among programs that can't necessarily talk to one another directly.

What is a JSON file

A JSON (JavaScript Object Notation) file is an open standard file format and data interchange format that uses human-readable text to store and transmit data objects consisting of attribute-value pairs and arrays (or other serializable values).
